# DocScan Pro

A professional-grade document scanning and PDF optimization tool created entirely with AI.

> **Disclaimer:** This application was completely built using AI by **me**. Only the application idea is mine. I am not a developer; I created this for personal use only.

## 🌟 Features

- **Mobile-First Scanning**: Use your device camera to capture high-quality document scans.
- **Perspective Correction**: Intelligent 4-point corner adjustment to fix skewed or angled scans.
- **Smart Rotation**: Correct document orientation with 90-degree rotation controls.
- **Aspect Ratio Preservation**: Automatically maintains the original proportions of your documents (A4, receipts, ID cards, etc.).
- **Advanced Export Modes**:
  - **By Size**: Specify a target file size (KB/MB) and the app automatically optimizes quality and resolution to fit.
  - **By Quality**: Manually set the visual fidelity (10% - 100%) for maximum control.
- **Multi-Page Support**: Scan multiple pages and export them as a single, optimized PDF or individual JPEGs.
- **PDF Import**: Upload existing PDF files to re-optimize or extract pages.

## 🚀 How to Run Locally

### Prerequisites
- [Node.js](https://nodejs.org/) (v18 or higher)
- npm (comes with Node.js)

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/YOUR_USERNAME/docscan-pro.git
   cd docscan-pro
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```
